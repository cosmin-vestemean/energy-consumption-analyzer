Complete parameter configuration for photovoltaic panel recommendations

üéØ What is the parametrizable system?

The parametrizable system allows complete customization of all values used in photovoltaic system sizing calculations. Instead of fixed (hardcoded) values, you can now adjust parameters to reflect:

  • Current market prices
  • Specific equipment specifications
  • Regional conditions (sunshine hours, irradiation)
  • Your personal autonomy requirements
  • Different investment scenarios

üìä E.ON Romania Reference Data

All default values are based on validated data from E.ON Romania, extracted from the official article: "How many photovoltaic panels does a house in Romania need?"

üìå Key Values for Romania

  • Average annual consumption: 2,500 kWh/year for a household
  • Standard panel power: 415W (market range: 250-550W)
  • Maximum solar hours: 4.5 hours/day (Romania average)
  • Production ratio: 1.3-1.6 (depending on location)
  • Panel surface: ~2 m¬≤ per panel

üöÄ How to Use the Configuration

Configuration Steps

1Open the Configuration Panel

  • Navigate to the "Photovoltaic System Recommendations" section
  • Look for the purple panel "‚öôÔ∏è Configure Photovoltaic System Parameters"
  • Click the "Show Advanced Configuration" button

2Modify Desired Parameters

  • Solar Panels: Adjust power, efficiency, solar hours
  • Batteries: Configure autonomy and costs
  • Inverter: Set safety margin and prices
  • Financial: Important! Update electricity price

3Automatic Validation

  • The system validates values in real-time
  • You'll see warnings if values are outside normal ranges
  • For example: a 150W panel will generate a warning (below 250W standard)

4Apply Configuration

  • Click "‚úî Apply Configuration" to recalculate recommendations
  • All charts and calculations update instantly
  • You'll see the new values in all sections

5Review Results

  • Number of panels needed
  • Recalculated battery capacity
  • Updated total costs
  • New investment payback period
  • Estimated annual savings

üí° Use Cases

1. Market Price Update

Situation: Energy prices changed from 0.80 to 0.95 RON/kWh

Solution:

  1. Open the configuration
  2. Modify "Electricity Price" ‚Üí 0.95 RON/kWh
  3. Apply configuration
  4. Payback period will decrease (greater savings)

2. Regions with Different Irradiation

Situation: You live in northern Romania (less sun)

Solution:

  1. Adjust "Maximum Solar Hours/Day" ‚Üí 4.0 hours
  2. Modify "Production Ratio" ‚Üí 1.2
  3. The system will recommend more panels to compensate

3. Off-Grid System

Situation: You want complete grid independence

Solution:

  1. Modify "Autonomy Days" ‚Üí 4 days
  2. Adjust "Safety Margin" ‚Üí 50% (1.5)
  3. The system will size larger batteries and inverter

4. Equipment Comparison

Situation: Comparing 415W vs 550W premium panels

Solution:

  1. Configure first variant (415W, cost 3.98 RON/W)
  2. Note the results (screenshot or notes)
  3. Switch to 550W, cost 4.50 RON/W, efficiency 90%
  4. Compare the two scenarios

‚ö†Ô∏è Important to Remember

Important Warnings:

  • These are estimates: Always consult an ANRE-authorized installer for exact calculations
  • Don't underestimate costs: 30% installation cost is a minimum, can be more
  • Check warranties: Ensure parameters match actual equipment
  • Degradation over time: Panels lose 0.5-1% efficiency per year

‚úì Recommendations:

  • Use local data: Adjust solar hours for your region
  • Update prices: Periodically check market costs
  • Consult E.ON table: Use as benchmark for validation
  • Oversize: Consider future growth (electric car, etc.)
